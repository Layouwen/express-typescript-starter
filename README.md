# Express Typescript Starter

## Description

This is a starter project for Express with Typescript.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run dev

# production mode
$ pnpm run build
$ pnpm start
```

## Test

```bash
# unit tests
$ pnpm run test
```

## Pkg Target

### Linux alpine

```bash
node16-alpine-x64
```

### MacOS

arm64

```bash
node16-darwin-arm64
```

x64

```bash
node16-darwin-x64
```
